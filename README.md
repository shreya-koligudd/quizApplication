# Quiz Application

A Java-based desktop Quiz Application with admin panel, user login, and score tracking built using Java Swing and MySQL.

## Features
- User Login and Registration
- Quiz with Multiple Choice Questions
- Score Calculation and Display
- Admin Panel to manage questions
- Database connectivity using MySQL

## Tech Stack
- Java (Swing / AWT)
- MySQL
- JDBC
- NetBeans IDE

## Setup Instructions

### Prerequisites
- Java JDK 8 or above
- MySQL Server
- NetBeans IDE (recommended)

### Steps
1. Clone the repository
 git clone https://github.com/shreya-koligudd/quizApplication.git
2. Create a MySQL database named `quizapp`
3. Copy `config.properties.example` to `config.properties`
4. Update `config.properties` with your MySQL credentials
5. Open the project in NetBeans and run it
